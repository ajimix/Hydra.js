Class Detail
ErrorHandler()
ErrorHandler is the class that will handle the errors on your code. Can be changed using setErrorHandler method. Default ErrorHandler will log messages in console if exist. If console doesn't exist the error will be logged in a hidden layer.
Author: Tomas Corral Casas.

log is the method that will differenciate the system if they had console or not. if windowdow.console exist console.log will be called if windowdow.console not exist then the log on layer will be activated.
